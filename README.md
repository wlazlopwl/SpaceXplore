# SpaceXplore

An Android application that allows you to view SpaceX data.
App using [SpaceX-API](https://github.com/r-spacex/SpaceX-API)

## Technologies
Project is created with:
* Kotlin
* Navigation Component
* Coroutines
* Retrofit
* Room database
* LiveData
* ViewModel, MVVM architecture
* Dagger2
* GSON
* Glide
* DataStore
